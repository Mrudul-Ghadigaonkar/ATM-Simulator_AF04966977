ATM Simulator
A simple, interactive command-line interface (CLI) application that simulates basic bank transactions.

Features
Secure Access: PIN-protected login (Default: 1234).

Transactions: Deposit and withdraw funds with real-time balance updates.

Validation: Prevents negative inputs and enforces a ₹10,000 transaction limit.

Statement: View a numbered history of all deposits and withdrawals.

How to Run
Ensure you have Python 3 installed.

Run the script:

Bash
python atm_simulator.py
Follow the on-screen prompts to manage your account.

Configuration
Initial Balance: ₹10,000

Max Transaction Limit: ₹10,000

Default PIN: 1234
